# Deep Research Plan: Нормативные базы и референсные значения

## Контекст

На основе исследования протоколов (answers/2.md) выявлена необходимость создания комплексной базы нормативных значений для различных популяций.

## Цели исследования

### 1. Возрастные и половые нормативы

**Задачи:**
- VO2max reference values по возрасту/полу
- Percentile charts development
- Z-score calculations

**Запросы:**
1. VO2max reference values age sex percentile charts
2. Cardiorespiratory fitness normative data population
3. FRIEND registry VO2max reference equations
4. Fitness Assessment and Exercise Testing normative values
5. Age-predicted VO2max equations comparison accuracy

---

### 2. Спортивные нормативы по видам

**Задачи:**
- Elite vs recreational athlete benchmarks
- Sport-specific VO2max requirements
- Performance level classification

**Запросы:**
1. Elite athlete VO2max values by sport discipline
2. Professional cyclist VO2max watt kg benchmarks
3. Endurance athlete physiological profiles database
4. Olympic athlete cardiorespiratory fitness data
5. Sport-specific lactate threshold power heart rate norms

### 3. Педиатрические референсные данные
